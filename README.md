# 2048Game
2048游戏

## 项目概述
项目支持游戏刷新，分数统计，基本的2048游戏功能，且带有合成动画炫酷效果

项目包含游戏成功，游戏失败机制...

项目编写的逻辑，作者自认为清新脱俗，且有一点抽象。看不懂可以联系作者，作者必将热情讲解它的来龙去脉

最值得一提的是，此项目监听了页面的获焦与失焦的事件，当页面失焦时会触发某些操作
